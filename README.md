# Forest Cover Monitoring

This project detects vegetation (tree crowns) using HSV masking and contour extraction.

## Steps
- Upload forest images
- Convert to HSV
- Apply green mask
- Extract contours
- Calculate vegetation pixels and percentage coverage

## Tools Used
- Python
- OpenCV
- NumPy
- Google Colab

## Output
- Vegetation mask
- Contour overlay
- Vegetation coverage (%) comparison
